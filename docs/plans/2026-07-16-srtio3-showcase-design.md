# SrTiO3 showcase design

Status: approved on 2026-07-16.

## Goal

Add a short Typst-native introduction to cubic strontium titanate, SrTiO3, as
the narrative showcase for `materia:0.1.0`. The document should teach a useful
piece of materials science while demonstrating the package's real-space,
reciprocal-space, band-panel, and scene-composition APIs. It is documentation,
not a first-principles calculation.

The requested formula is interpreted as SrTiO3, matching the existing
perovskite example. SiTiO3 is out of scope.

## Audience and tone

The primary reader knows undergraduate solid-state physics or materials
science but has not used `materia`. The text should be compact and scientific,
with claims tied to primary literature. Code listings must be copyable and the
figures must be generated by the code shown. Avoid tutorial filler and avoid
implying that `materia` computes phonons or electronic bands.

## Narrative

The document will occupy three A4 pages.

### Page 1: From formula to coordination geometry

Introduce the cubic perovskite reference structure in space group Pm-3m with
`a = 3.905 angstrom`: Sr on 1a, Ti on 1b, and O on 3c. A large unit-cell view
will show atoms, Ti-O bonds, the TiO6 coordination octahedron, crystallographic
axes, and a legend. A smaller supercell view will make the corner-sharing
octahedral network visually obvious.

The page demonstrates `prototypes.perovskite`, `crystal-scene`, `crystal`, and
ordinary `scenery` composition. Text will distinguish this cubic reference
from the low-temperature tetragonal phase rather than presenting the cubic
cell as the zero-temperature ground-state structure.

### Page 2: The same periodicity in reciprocal space

Map the primitive-cubic lattice to its first Brillouin zone and the connected
path Gamma-X-M-Gamma-R-X. Place the zone beside a schematic band panel. The
panel will mark the valence-band maximum at R, the conduction-band minimum at
Gamma, and an indirect gap of approximately 3.25 eV. A short annotation will
identify the valence edge as predominantly O 2p and the conduction edge as Ti
3d t2g.

The geometry and path are produced by `bz-scene`/`bz-figure`, `band-axis`, and
`band-panel`. The band curves are analytic illustration data supplied by the
document and will be labelled "schematic" both on the panel and in the prose.

### Page 3: What is computed, and what is supplied

Show concise code excerpts beside a data-flow summary:

`space group + Wyckoff sites -> expanded structure -> bonds/polyhedra -> scene`

and

`lattice -> reciprocal basis -> Wigner-Seitz cell -> k-path`

A final table will separate `materia` outputs from user inputs. A short
scientific sidebar will connect SrTiO3 to the 105-110 K antiferrodistortive
transition, soft polar modes, and quantum paraelectricity, without attempting
to visualize a calculated phonon dispersion.

## Data and rendering architecture

The main source will be `materia/examples/srtio3-introduction.typ`. Reusable
values such as the structure, lattice parameter, path labels, analytic bands,
and colours will be defined once near the top. Figures and displayed snippets
will consume those values so numeric claims cannot drift between prose and
graphics.

Figures use ordinary direct Typst calls. Copyable snippets show the same public
API without source strings, injected scopes, or `eval`, keeping the article close
to how a scientist would naturally use `materia` in a document.

A dedicated compile-time test will assert:

- five atoms in the primitive cubic cell;
- the selected path endpoints and sample counts;
- equal sample counts for every schematic band;
- the intended R-to-Gamma indirect-gap annotation.

No external data file is needed. The publication preview will be the rendered
first page at `materia/images/srtio3-introduction.png`; the full three-page
source remains linked from the README and excluded from the downloadable
archive together with other examples and images.

## Scientific sources and claim boundaries

The article will cite primary sources, using only short paraphrases:

- G. Shirane and Y. Yamada, "Lattice-Dynamical Study of the 110 K Phase
  Transition in SrTiO3," Physical Review 177, 858 (1969),
  https://doi.org/10.1103/PhysRev.177.858.
- A. Janotti, D. Steiauf, and C. G. Van de Walle, "Strain effects on the
  electronic structure of SrTiO3," Physical Review B 84, 201304(R) (2011),
  https://doi.org/10.1103/PhysRevB.84.201304.
- D. Shin et al., "Quantum paraelectric phase of SrTiO3 from first
  principles," Physical Review B 104, L060103 (2021),
  https://doi.org/10.1103/PhysRevB.104.L060103.

The document must explicitly distinguish:

- crystallographic geometry computed by `materia`;
- conventional path geometry tabulated from published standards;
- schematic electronic curves authored for the figure;
- literature values and interpretations quoted in the prose.

## Failure handling

The dedicated regression test should fail at compile time if band arrays and
path samples have different lengths or if the expected perovskite structure is
not produced. All package calls use public APIs. No generated PDF, test output,
cache, or build script will enter the Universe package directory.

## Verification and release

Before updating the `materia` submission:

1. Compile the three-page document with Typst 0.15.
2. Render and visually inspect every page, including the first-page README
   preview.
3. Run `make test` and `make examples` from the monorepo root.
4. Run Markdown link checks.
5. Stage the minimal `materia:0.1.0` directory beside the merged
   `scenery:0.1.0` package and run `typst-package-check` online.
6. Open the `materia:0.1.0` Universe PR only after every gate is green.
